# Empowered Care - Outbreak Detection System

AI-powered multi-agent disease outbreak detection system for high-value health systems.

## Tech Stack

- **Backend**: FastAPI (Python)
- **Frontend**: Vite + React + Tailwind CSS
- **AI**: Gemini Multi-Agent System

## Getting Started

### Local Development

1. **Backend**:
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```

2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
